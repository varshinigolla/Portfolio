My Projects

Taco-DB: High-Performance Relational Database System Development

As part of a comprehensive academic project, I spearheaded the development of Taco-DB, a relational database management system (DBMS) built from the ground up in C++. This project aimed to tackle challenges in efficient data storage, indexing, and query processing, providing an optimized solution for modern database applications. The development process involved designing and implementing core DBMS components, starting with a slotted page structure for streamlined storage management and a buffer manager to enhance I/O efficiency. Leveraging POSIX I/O, I ensured optimal file handling, which significantly improved data manipulation and retrieval speeds. In pursuit of efficient query processing, I implemented key operators such as selection, aggregation, and external merge sort, each with corresponding execution states. To further boost performance, I designed and implemented a B-tree indexing system, enabling optimized data retrieval through support for variable-length keys, node splitting/merging, and fast search/scan operations. This approach ensured faster and more scalable query performance, even across large datasets.

I experimented with various join algorithms, including nested-loop join, sort-merge join, and index nested-loop join, ultimately incorporating iterator management and rewind functionalities for seamless execution. My ability to evaluate and optimize these algorithms ensured efficient relational operations and reduced query execution times. This experience not only sharpened my expertise in database systems, query optimization, and advanced data structures but also provided me with hands-on exposure to low-level system programming. It reinforced my passion for solving complex problems by combining theoretical knowledge with practical implementation, preparing me for future challenges in database and system-level development.

Smart CCTV Surveillance System

The Smart CCTV Surveillance System is a cutting-edge project developed using Python, OpenCV, and computer vision technologies to revolutionize traditional security systems. This system integrates features such as anti-theft detection, motion and noise tracking, visitor counting, and facial recognition, all supported by a sleek and user-friendly GUI built with Tkinter. Python was chosen for its simplicity, vast library support, and suitability for machine learning and computer vision tasks. OpenCV serves as the backbone of the project, offering powerful algorithms for image processing and object detection, enabling the system to perform real-time surveillance tasks with high accuracy. The project employs robust methodologies to ensure functionality and efficiency. Anti-theft detection leverages frame comparison techniques like SSIM to identify stolen objects, while motion detection monitors pixel changes to highlight unauthorized movements. Facial recognition uses Haar cascade classifiers and LBPH algorithms to detect and identify individuals from a pre-trained dataset. Visitor tracking records entry and exit activities with timestamps, while live video streaming allows remote monitoring. A systematic waterfall model guided the development process, ensuring structured execution and thorough testing through white-box and black-box techniques. Real-time alerts and analytics are generated to provide actionable insights, addressing challenges such as false positives, privacy concerns, and accuracy issues effectively.

This system eliminates the need for constant human monitoring.

Blogging Platform with User Authentication

The Blogging System is designed for content creators and readers, enabling seamless blogging experiences with a strong emphasis on user interaction and security. This system incorporates user authentication using JWT (JSON Web Tokens) to ensure secure access and user-specific functionalities. Registered users can create, edit, and delete their blog posts, with additional features like tagging, categorization, commenting, and liking posts to enhance engagement and user interaction. The system’s architecture is built on a ReactJS frontend, a Node.js backend, and a MongoDB database, offering scalability, efficiency, and responsiveness. With tagging and categorization, posts are organized for easy discovery, and a robust commenting system allows users to interact with blog authors, fostering a sense of community. The inclusion of a liking feature promotes engagement and enables readers to highlight quality content. The ReactJS-based frontend ensures a dynamic, responsive user experience, while the backend, powered by Node.js and Express, efficiently handles API requests and business logic. MongoDB serves as the database, storing structured and unstructured data such as posts, comments, and user details. To ensure security and scalability, the system uses JWT authentication, protecting endpoints and ensuring secure user sessions. The backend implements role-based access controls, ensuring that only authorized users can perform sensitive actions like editing or deleting posts.

Student Salary Price Prediction

The Student Salary Prediction project was developed under the guidance of Dr. Shamshad Parveen to predict student salaries based on key features derived from historical data. Using NumPy and pandas, the dataset was meticulously analyzed and preprocessed to ensure accuracy and relevance of the input features. The platform provides an intuitive interface for interacting with the model, allowing students and educators to predict salaries effectively. This project showcases proficiency in machine learning and practical deployment, along with strong technical skills in Python, NumPy, pandas, and frontend development with HTML and CSS.
